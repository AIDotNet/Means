title: Metrics Reference
description: "Core Prometheus metrics exposed by `/metrics`."
---

Means exposes Prometheus text metrics:

```text
GET /metrics
```

## Core Metrics

| Metric | Meaning |
| --- | --- |
| `means_storage_buckets` | current bucket count |
| `means_storage_objects` | current visible object count |
| `means_storage_object_bytes` | current logical object bytes |
| `means_cluster_capacity_bytes{state}` | total/available/used capacity |
| `means_cluster_nodes{status}` | online/offline node count |
| `means_cluster_disks{status}` | online/offline disk count |
| `means_replica_repair_queue{status}` | replica repair backlog |
| `means_background_task_failure_count{task}` | background task failures |
| `means_background_task_status{task,status}` | current background task state |

## Scrape Example

```yaml
scrape_configs:
  - job_name: means
    metrics_path: /metrics
    static_configs:
      - targets:
          - localhost:5178
```
